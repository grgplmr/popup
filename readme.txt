Plugin WordPress pour créer des pop-ups avec effet glassmorphism - pop-up d'accueil et exit intent personnalisables.

== Description ==

Pop-up Glassmorphism est un plugin WordPress moderne qui vous permet de créer facilement des pop-ups élégants avec un effet de verre translucide (glassmorphism). 

**Fonctionnalités principales :**

* **Pop-up d'accueil** : Affichage automatique après un délai personnalisable
* **Pop-up Exit Intent** : Détection de l'intention de quitter la page
* **Design Glassmorphism** : Effet de verre moderne et élégant
* **Interface d'administration complète** : Configuration facile dans le back-end WordPress
* **Éditeur riche** : Insertion de texte formaté et de liens cliquables
* **Personnalisation avancée** : Couleurs, tailles, transparence
* **Prévisualisation en temps réel** : Voir le rendu avant publication
* **Responsive** : Compatible mobile et desktop
* **Accessible** : Navigation clavier et focus visible
* **Optimisé** : Code léger sans dépendances lourdes

**Compatibilité :**

* Compatible avec tous les thèmes WordPress
* Testé avec GeneratePress et Gutenberg
* Aucune dépendance jQuery
* Support multilingue

== Installation ==

1. Téléchargez le plugin et décompressez-le
2. Uploadez le dossier `popup-glassmorphism` dans `/wp-content/plugins/`
3. Activez le plugin via le menu 'Plugins' de WordPress
4. Allez dans 'Pop-up Glassmorphism' dans le menu d'administration
5. Configurez vos pop-ups selon vos besoins

== Frequently Asked Questions ==

= Le plugin fonctionne-t-il avec tous les thèmes ? =

Oui, le plugin est conçu pour être compatible avec tous les thèmes WordPress. Il a été spécialement testé avec GeneratePress.

= Les pop-ups sont-ils responsive ? =

Oui, les pop-ups s'adaptent automatiquement aux différentes tailles d'écran (mobile, tablette, desktop).

= Puis-je personnaliser les couleurs ? =

Absolument ! Vous pouvez personnaliser les couleurs de fond (avec transparence), la couleur du texte, et bien plus encore.

= Le plugin affecte-t-il les performances ? =

Non, le plugin est optimisé pour être léger et n'utilise aucune dépendance lourde comme jQuery.

= Comment fonctionne la détection d'exit intent ? =

Sur desktop, la détection se fait quand la souris quitte la fenêtre par le haut. Sur mobile, elle se déclenche lors d'un scroll rapide vers le haut.
